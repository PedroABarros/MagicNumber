# 🎯 Magic Number

Magic Number is a terminal-based guessing game built with Python, where the player must guess a randomly generated number between 1 and 100.

The game provides dynamic feedback to guide the player, including directional hints (higher/lower) and proximity indicators (hot or cold). It also features a scoring system, difficulty modes, and a persistent leaderboard stored locally.

This project was created as a way to practice Python fundamentals, including control flow, file handling, input validation, and modular code structure.

## ⚙️ Features

🎯 Random number generation

🔥 Hot / Cold proximity hints

🔼 Higher / Lower directional hints

🏆 Top 5 leaderboard stored locally

📊 Score system based on remaining attempts

🎨 ANSI colored terminal interface

📉 Dynamic search range adjustment
